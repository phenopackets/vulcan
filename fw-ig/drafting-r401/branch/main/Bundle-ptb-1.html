Bundle: PhenoTips example bundle 1

This Bundle is meant to capture the PhenoTips sample data and show validation results according to the base FHIR R4 specification and the GA4GH Phenopacket (version 2) FHIR Implementation Guide. It currently only has a copy of DevelopmentalDely example of the Phenopacket Composition profile listed here as a starting point and a placeholder to be replaced by the information from the PhenoTips data files.

See this issue for discussions related to this example.
